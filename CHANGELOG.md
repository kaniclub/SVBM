# 更新履歴

このファイルでは、BomberGear の主な更新内容をまとめます。

## [1.0.1] - 2026-03-11

### 改善
- 石系アイテム判定を調整
- ID に加えて名前でも判定するようにし、一部の石系アイテムが爆風対象から漏れる問題を軽減

## [1.0.0] - 2026-03-11

### 追加
- ボンバーマン風の専用装備 **BomberGear** を追加
- ソケットによるカスタマイズ機能を追加
- 爆弾特性を追加
  - リモコン爆弾
  - 貫通爆弾
- アクション特性を追加
  - キック
  - 持ち上げ＆投げるアクション
- 火力強化、同時設置数強化を追加
- ソケットメニューを追加
  - 現在の装備状態の確認
  - ソケットの取り外し
  - 火力、爆弾数、特性の確認
  - ツルハシ / オノ段階の表示
- ソケットアイテムのドロップ機能を追加
- 郵便による BomberGear 支給機能を追加
- `config.json` および GMCM による設定変更に対応
