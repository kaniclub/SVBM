# BomberGear

**BomberGear** は、Stardew Valley に**ボンバーマン風の爆弾アクション**を追加する SMAPI Mod です。  
専用装備 **ボンバーギア** にソケットを装着して、爆弾の性能やアクションを強化できます。

## 必要環境

- SMAPI
- Content Patcher
- Generic Mod Config Menu（任意）

## 主な機能

- 専用装備 **ボンバーギア** を追加
- 爆弾を足元に設置
- 十字方向に爆風が広がる
- ソケットで爆弾性能を強化
- ソケットメニューで現在の状態を確認・取り外し可能

## ソケット一覧

| 種類 | 効果 |
|---|---|
| リモコン爆弾チップ | 設置済み爆弾を右クリック短押しで起爆 |
| 貫通爆弾チップ | 爆風が壊せる対象で止まりにくくなる |
| キックチップ | 爆弾を蹴って滑らせる |
| スローチップ | 爆弾を持ち上げて投げる |
| パワーコア | 爆弾の火力を強化 |
| ボムバッグ | 同時に置ける爆弾数を強化 |

## 基本操作

| 操作 | 内容 |
|---|---|
| 左クリック / ツール使用ボタン | 爆弾を設置 |
| 右クリック / アクションボタン | 特殊アクション |
| `K` キー（初期設定） | ソケットメニューを開く |

## ソケットメニューで確認できる内容

- 爆弾特性
- アクション特性
- 火力
- 同時設置数
- ツルハシ段階
- オノ段階

## 入手方法

- **ボンバーギア本体** は郵便で支給されます
- **ソケットアイテム** は設定により、石や鉱石ブロックの破壊時にドロップします

## 設定できる内容

- 導火線時間
- 初期火力
- 火力上限
- 初期同時設置数
- 爆弾数上限
- プレイヤー / モンスターダメージ
- キック速度
- 戦闘不能時のソケット初期化
- ソケットドロップ関連設定

## 注意

- 爆弾は設定次第でプレイヤー自身にもダメージがあります
- ソケットの差し替えや取り外しには、インベントリ空きが必要な場合があります
- ソケット構成によって操作感が変わります
